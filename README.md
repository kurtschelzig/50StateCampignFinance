Goal: The goal of this project is to export Campaign contribution data from all 50 states into CSVs

Requirments: Docker with a Firefox Selenium image, R

How To:

1.Set Working Directory to desired file destination. This will create alot of folders so make sure it is in an empty directroy.
2. Run the DirectorySetup.R program in your desired directory.
3. Copy and Paste the generated PowerShell Script into Powershell and run. This will create the Docker instance
4. Run the 50StatesExportScript.
5. To fetch the data from a state for a given year, simply run 'Name Of State'(Year)
